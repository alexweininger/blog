My First Chrome Extension

Function
2018-02-21

I have always loves Chrome extensions, giving me one more thing to customize, and hopefull that much more features and tools. Since I have been using a lot of Javascript recently, I have become more an dmore comfortable with its workings. I decided I didn’t have enough on my plate and started looking at the Chrome API. This was the first project where I was able to interperet and use API documentation to build something. I’ve finally gotten to the point where I can consistenly read and comprehend documentation, which felt great.

My extension was going to list recently viewed PDF files in a pop-up window. Useful to me as a student who often ends up downloading the same PDF every time I need to access it rather than going to my downloads folder and opening it from there. I would find out later that accessing local files via Chrome extension is impossible because it’s a security threat (duh).
